### Deploy a Template

```bash
# Clone this repo
gh repo clone [your-username]/code-notes

# Pick a template
cd templates/personal-knowledge-base

# Deploy to new GitHub repo
./deploy_me.sh
```

Available templates:
- Personal Knowledge Base (Zettelkasten system)
- Goal Tracker & Planning (Proactive daily planning)
- Language Learning Assistant
- Research Automation
- Self-Learning Repository
- Autonomous Dev Environment
- Project Scaffolder

1. **Start Claude Code session** in this repository

   ```

   I need help with: [your goal]
   ```
3. **Claude will:**
   - Ask clarifying questions

   - Build customized templates
